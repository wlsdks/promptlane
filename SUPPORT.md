# Support

promptlane is pre-release open-source software.

## Questions And Ideas

Use GitHub Discussions for:

- setup questions
- workflow ideas
- adapter requests
- prompt improvement feedback
- benchmark and evaluation discussion

Do not paste real prompts, private paths, credentials, API keys, OAuth tokens,
session tokens, or proprietary source code into public discussions.

## Bugs

Use GitHub Issues for reproducible bugs. Include:

- promptlane version
- operating system
- Node.js version
- command or UI path
- expected behavior
- actual behavior
- redacted logs, if needed

## Security

Do not open a public issue for security reports. Follow `SECURITY.md`.
